# sound_recoding_project
voice or sound recording project
This project is a simple voice recording application built using Python. It allows the user to record audio for a selected duration and automatically saves the recorded sound as a WAV file. It is ideal for beginners learning about audio processing and Python libraries.

✨ Features
Records audio using your device’s microphone
User can enter the duration of the recording
Saves audio output automatically as out.wav
High-quality recording using a 44.1 kHz sample rate
Lightweight and easy to run

What the Program Does
Asks the user for the duration of the recording
Starts capturing audio from the microphone
Waits until the recording is complete
Saves the audio file as out.wav
Notifies the user when the process is finished
